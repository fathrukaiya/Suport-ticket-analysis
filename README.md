# Support Ticket Analysis Dashboard

## 📌 Project Overview
This project analyzes support ticket data using SQL, Excel/CSV, and Power BI to identify trends, recurring issue categories, SLA risks, backlog growth, and overall service performance.

The objective is to help support teams improve response times, reduce repeated incidents, and make data-driven operational decisions.

---

## 🛠 Tools Used
- SQL Server
- Excel / CSV
- Power BI
- GitHub

---

## 📂 Project Structure
- `sql/queries.sql` → SQL analysis queries
- `data/tickets.csv` → support ticket dataset
- `excel/excel_dashboard.xlsx` → Excel pivot dashboard
- `dashboard/powerbi_screenshots.png` → dashboard screenshots
- `README.md` → project documentation

---

## 🎯 Business Questions Solved
- What are the most common ticket categories?
- Which tickets are still unresolved?
- What is the daily ticket volume trend?
- Which priorities remain open the longest?
- Which agents close the most tickets?
- Which modules have the highest average resolution time?

---

## 📈 Key Insights
- Identified top recurring issue categories driving ticket volume
- Measured unresolved ticket backlog and open ticket trends
- Highlighted high-priority tickets with delayed resolution
- Tracked daily support demand patterns
- Compared module-wise SLA performance
- Improved visibility into support operations KPIs
